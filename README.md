 Interactive Sales Dashboard

 Project Overview
This project analyzes sales data using Seaborn and Plotly.
The dashboard shows:
- Sales distribution by product
- Quantity distribution
- Correlation heatmap
- Sales trend over time
- Region-wise sales performance
 Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Plotly

 Setup Instructions
1. Install Python 3.10+
2. Install libraries:
   pip install -r requirements.txt
3. Run:
   python dashboard.py

 Features
- 5+ different chart types
- Statistical visualization
- Interactive Plotly chart
- Professional dashboard layout

## Conclusion
The dashboard helps understand product performance,
regional sales distribution, and overall trends.
